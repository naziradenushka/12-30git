Тема урока: ветки и комиты

  • Создание ветки - git branch namebranch(название ветки)
  • Слияние веток - git merge namebranch(название ветки)
  • Переход на коммит или переход на ветку - git checkout
  • Удаление веток - git branch -d имя ветки
  • Создание ветки и сразу же переход на созданную ветку - git checkout -b имя ветки
